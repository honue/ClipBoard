# ClipBoard
## 功能
1.剪切板历史 快捷键 command shift v

2.跨设备内容自动粘贴(iPhone复制,mac自动粘贴内容到编辑处)

> 自己是用来刷题做笔记,手机复制题目解析,mac自动粘贴到Typora,快速做笔记.
